This is an art bookmark app using Rijks museum api

# Getting Started
## Step 0: .env 
```bash
cp .env.example .env
```
Insert provided env variables

## Step 1: Install Node modules
```bash
yarn install
```
## Step 2: Install Pods
```bash
cd ios
pod install
```
## Step 3: Start project
### For iOS
```bash
# Navigate to root of the project and run:
yarn ios
# OR using Xcode
xed ios 
```
### For Android
```bash
# Navigate to root of the project and run:
yarn android
```
